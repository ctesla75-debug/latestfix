Blood Sugar Tracker (Meals Edition) – Install as an App (Android)

1) Host this folder on an HTTPS site (Netlify is easiest).
2) Open the link in Chrome on your phone.
3) Menu (⋮) -> Install app (or Add to Home screen).

Offline:
- Open once online so it caches, then it works offline.
- Data is stored locally on your phone.


AU settings in this build:
- Waist: mm
- Weight/Grip: kg
- Dates displayed/exported as dd/mm/yyyy (internally stored as YYYY-MM-DD).
